.. _install:

How to install
===============

.. toctree::
   :maxdepth: 2

Clone the repository and run

.. code-block:: bash

    pip install .

Prerequisites
-------------

Python 3.x and Julia 1.x with packages StatsBase.jl, LinearAlgebra.jl, CorrelationFunctions.jl (version=0.11.0)
and EulerCharacteristic.jl should be installed.

OS compatibility
----------------

PARSE is devepoled for Linux, we do not guarantee the correct operating of this library in other OS.
